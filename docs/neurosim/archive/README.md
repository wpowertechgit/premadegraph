## Használati szabály

Innen csak akkor dolgozz, ha:

- elveszett kontextust kell visszanyerni;
- össze kell hasonlítani régi és új dokumentumverziót;
- meg kell érteni, hogy egy döntés honnan ered.

Ne használd elsődleges thesis-forrásként. Keress helyette stabilabb dokumentumot a `architecture/`, `mechanics/`, `validation/` vagy `chapter-writing/` mappákban.
